Postmate Client is a **Postman alternative for Visual Studio Code** designed for developers who want **fast, local, and privacy-first API testing** without cloud sync or login.

## Why Postmate Client?

- ✅ Privacy-first API testing (no cloud sync, no login)
- ✅ Built natively for Visual Studio Code
- ✅ Fast and lightweight (no heavy desktop apps)
- ✅ Full support for collections, environments, and variables
- ✅ Powerful data-driven testing with CSV support
